# Clean Artifacts

You are cleaning up temporary artifacts generated by Grimoire rites across Arcana and all installed domain grimoires.

## Steps

1. Run the cleanup rite:

```bash
python3 {{ARCANA_PATH}}/rites/clean_artifacts.py
```

On Windows, use `python` instead of `python3`.

2. Report the output to the user — show what was cleaned.

3. If the user wants to preview without deleting, run with `--dry-run`:

```bash
python3 {{ARCANA_PATH}}/rites/clean_artifacts.py --dry-run
```
